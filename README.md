# Expense Tracker

**Автор:** [Дима Криницын]

## Описание
Простое десктопное приложение для контроля личных финансов. Позволяет записывать траты, классифицировать их и просматривать общую сумму по фильтрам.

## Инструкция по использованию
1. Запустите файл `expense_tracker.py` через Python 3.
2. Введите сумму расхода (число).
3. Выберите категорию из выпадающего списка.
4. Укажите дату (по умолчанию стоит текущая).
5. Нажмите "Добавить расход". Данные автоматически сохранятся в файл `expenses.json`.
6. Используйте блок фильтрации для просмотра расходов по конкретным категориям.

## Тестирование
- Попробуйте ввести отрицательную сумму или буквы — приложение выдаст ошибку.
- Попробуйте оставить поле категории пустым — приложение выдаст предупреждение.
